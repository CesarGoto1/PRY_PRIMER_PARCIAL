# MagicTouring

Autor: César González

Instrucciones de ejecución:
1. Abre este proyecto en Visual Studio Code.
2. Instala y habilita la extensión "Live Server".
3. Abre `index.html` (archivo en la raíz) y elige "Open with Live Server" para lanzar la aplicación en tu navegador.

Notas:
- El proyecto usa módulos ES (scripts con `type="module"`), por lo que debe servirse desde un servidor (por eso se recomienda Live Server).
- Si usas la consola del navegador, revisa `localStorage` y los eventos de las componentes web (por ejemplo, la clase `BookingModal` en `public/js/booking-modal.js`) para depuración.
